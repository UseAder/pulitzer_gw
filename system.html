CLASS FEATURES
班级特点
BASIC CLASS
基础班
情商课程培养四大目标：
自信心、人际交往、独立性、专注力
口才课程培养目标：
掌握语音发音基础训练，加强孩子口齿发音，语言表述流畅，能够清晰表达自我想法，培养专注力和自信心。克服紧张情绪全面释放天性，敢在众人面前讲话，大胆沟通，有效提高人际交往能力。
BASIC CLASS
提高班
情商课程培养四大目标：
情绪管理、同理心、挫折抵抗、创造力
口才课程培养目标：
熟练掌握语音发声，语流音变，语言表达标准、流畅，说话用词准确自如，
有良好的语言表达习惯。
可以自如的进行主持、朗诵、表演等，听说读写全面发展。提升逻辑、推理、
思考、判断等认知与发展能力，掌握良好的行为学习习惯。
BASIC CLASS
精英班
情商课程培养四大目标：
问题解决、责任心、感恩、领导力
口才课程培养目标：
说话具备演讲力和感染力，具有较好的语言逻辑思维，能自如表达个人观点，熟练掌握语言表达技巧。教学中将着力锻炼孩子即兴说话能力、即兴评述技巧和与人沟通的方式及情商社交能力，并重点进行演讲训练。提升主持、朗诵、表演、相声等综合训练。脱离自我中心，会用不同的观点看问题，初步建立社会观。班中学员从提高班中挑选，必须经过严格的面试选拔才能进入精英班。
